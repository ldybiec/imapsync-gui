<h1>Imapsync GUI</h1>
A simple graphical user interface (GUI) for <b>imapsync</b>, allowing convenient mailbox migration without using the command line, using only basic authentication.

<h2>Features</h2>

- Simple GUI for imapsync<br>
- Mailbox-to-mailbox migration<br>
- No command line required<br>
- Uses imapsync as backend<br>

<h2>Install dependencies</h2>
<pre>
nodejs 
npm 
imapsync
</pre>

<h2>Manual Installation</h2>
1. Create a project directory and extract or copy the downloaded files into it. 
<pre>mkdir /opt/imapsync-gui && cd /opt/imapsync-gui </pre>
2. Installing the Express library for HTTP server and routing. 
<pre>npm install express </pre>
3. Starting the server. 
<pre>node server.js</pre>
4. Open in browser.
<pre>http://localhost:3000</pre>
